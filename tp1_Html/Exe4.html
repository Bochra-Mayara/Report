Le Web
Système hypertexte public fonctionnement sur Internet et qui permet de consulter, avec un navigateur, des pages mises en ligne dans des sites
Les services Web
Les services de l'internet sont offertes par les serveurs présents sur l'internet:
  1. Web (documents hypertextes)
  2. Courrier électronique (e-mail)
  3. Messageries Instantanées
  4. Forums de discussion
  5. Téléchargement de fichiers
